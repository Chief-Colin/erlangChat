# erlangChat
erlang registration and chat system using mochiweb and mqtt broker (IOT)

TO SET UP THE APPLICATION, RUN:
-----------------------------------------------------
1.  git clone https://github.com/chiefcorlyns/erlangChat
2.  cd erlangChat
3.  mkdir db && mkdir db/dev
4.  make
5.  ./start-dev.sh
